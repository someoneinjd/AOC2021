# AOC2021

C++ solutions for Advent of Code 2021 (using [range-v3](https://github.com/ericniebler/range-v3)) 

## Build && Run

```shell
# Use the xmake build system
# https://github.com/xmake-io/xmake
xmake && xmake run
```

## Benchmark

```shell
xmake f --benchmark=y
xmake && xmake run
```

## Debug

```shell
xmake f -m asan
xmake && xmake run
```
